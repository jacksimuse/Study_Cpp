# Class

1. 구조체와 클래스
구조체는 타입이 다른 멤버를 하나의 범주 안에 모은 것. 데이터의 집합이어서 정보만 저장할 뿐 동작은 표현할 수 없다.
클래스는 멤버 변수와 함수까지 포함하여 속성뿐만 아니라 동작까지 표현할 수 있게 되었다.

구조체는 하위 호환성 유지를 위해 공개가 디폴트지만 클래스는 정보 은폐를 위해 가급적 멤버를 숨긴다.

2. 클래스의 특징
클래스 타입의 객체를 가리키는 포인터를 만들면 멤버 함수를 호출할 때는 -> 연산자를 사용한다.
C 문법이 클래스에 대해서도 일관되게 적용되어 이차 배열이나 이중 포인터도 선언할 수 있다.

정보를 저장하려면 클래스 타입의 변수를 선언해야한다.

3. 클래스 모듈
독립된 부품인 클래스는 보통 별도의 모듈로 작성하며 클래스의 이름과 같은 파일명을 사용한다.
Time 클래스를 만든다면 Time.h 헤더 파일에 클래스 선언문을 작성하고 멤버 함수의 본체는 Time.cpp 구현 파일에 작성한다.

-> 클래스는 독립 부품이며 필요할 때 하나씩 가져다가 합치면서 새로운 완성체를 만든다고 생각할 수 있다.
